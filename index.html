HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • href atribute points to the URL for the link
  • src: the location or source of the image file
  • alt: contains a text string that describes the apparance an functionally of the image

CSS

  • CSS determines the style of a webpage in order to make the website visually appealing
  • External CSS is the most commonly used, makes design changes easier
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • link - creates connection to external link
  • rel - specify the relationship between the current document and the linked document resource
  • href - specifies the location of the external resource

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

    • A variable is a named container that allows us to store data in our code.
    • Control flow is the order in which a computer executes code in a script.
    • JavaScript runs from top to bottom
    • An "if" statement is a way of introducing descicion making into our code
    • A function is a set of insturctions that tells the computer hot to perfom a certain task
    • 2 requirements for a function:
      1. Define a function
      2. Call a function
